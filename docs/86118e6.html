空间复杂度

发表于2021-02-16|更新于2021-03-22|LeetCode概念梳理

概念

一个算法的空间复杂度定义为该算法所耗费的存储空间，它也是问题规模 n 的函数。与时间复杂度类似，它反映的同样也是一个趋势。

常见的空间复杂度

如果算法执行所需要的空间不随规模 n 的大小而变化，那么其空间复杂度可表示为 O(1)O(1)O(1)。

如果算法需要的空间和规模 n 呈线性变化，那么其空间负责都可表示为 O(n)O(n)O(n)。
