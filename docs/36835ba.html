1808.好因子的最大数目

发表于2021-03-29|更新于2021-03-31|LeetCode题解1801-1900

题目信息

题目描述

给你一个正整数 primeFactors 。你需要构造一个正整数 n，它满足以下条件：

  1. n 质因数（质因数需要考虑重复的情况）的数目不超过 primeFactors 个
  2. n 好因子的数目最大化： 如果 n 的一个因子可以被 n 的每一个质因数整除，我们称这个因子是好因子。比方说，如果 n = 12，那么它的质因数为 [2,2,3]，那么 6 和 12 是好因子，但 3 和 4 不是。

请你返回 n 的好因子的最大数目。由于答案可能会很大，请返回答案对 109+710^9 + 7109+7 取余的结果。

请注意，质数的定义是大于 1，且不能被分解为两个小于该数的自然数相乘的数。一个数 n 的质因子是将 n 分解为若干个质因子，且它们的乘积为 n。

1≤primeFactors≤1091 \le primeFactors \le 10^91≤primeFactors≤109

示例输出

输入： primeFactors = 5
输出： 6

输入： primeFactors = 8
输出： 18

解题思路

纯数学题，本题和343. 整数拆分的数学逻辑是相同的。
代码实现中幂运算的部分参考50. Pow(x, n)。

假设对于某正整数 n，其质因子为：b1，b2，…\dots…，bm。每个质因子的个数为：a1，a2，…\dots…，am个。那么最终的好因子的个数等于：a1×a2×⋯×ama_1\times a_2\times \dots \times a_ma1​×a2​×⋯×am​。根据题意这个值的最大值就是最终的答案，与这个正整数的值，其因子的值实际上是无关的。

由算术-几何平均不等式：x1+x2+⋯+xnn≥x1×x2×⋯×xnn\dfrac {x_1 + x_2 + \dots +x_n} n \ge \sqrt[n]{x_1 \times x_2 \times \dots \times x_n}nx1​+x2​+⋯+xn​​≥nx1​×x2​×⋯×xn​​（当且仅当每个值都相等时等号成立），可得：当且仅当a1=a2=⋯=ana_1=a_2=\dots=a_na1​=a2​=⋯=an​时，a1×a2×⋯×ama_1\times a_2\times \dots \times a_ma1​×a2​×⋯×am​ 取得最大值。

假设将数字 n 以因子 x 等分为 a 份，那么 n=axn = axn=ax，其乘积为：xa=(x1x)nx^a={(x^{\frac 1 x})}^nxa=(xx1​)n，得到 x1xx^{\frac 1 x}xx1​ 需要取到最大值。

令：y=x1xy = x^{\frac 1 x}y=xx1​，对 yyy 两边同时取对数并求导得到：y′=1−ln⁡xx2x1xy'= \dfrac {1 - \ln x} {x^2} x^{\frac 1 x}y′=x21−lnx​xx1​。

令 y=0y = 0y=0，得到：1−ln⁡x=01 - \ln x = 01−lnx=0，所以x=ex = ex=e。由于：

y′{>0,x∈[−∞,e)<0,x∈(e,∞]y' \begin{cases} > 0, & x\in [-\infty, e)\\ < 0, & x\in (e, \infty] \end{cases} y′{>0,<0,​x∈[−∞,e)x∈(e,∞]​

所以 x=ex=ex=e 是 y=x1xy = x^{\frac 1 x}y=xx1​ 的极大值点。

但题目中需要取整数，与 eee 临近的整数是 3 和 4，y(3)=313≈1.44y(3) = 3^{\frac 1 3} \approx 1.44y(3)=331​≈1.44，y(4)=414≈1.41y(4) = 4^{\frac 1 4} \approx 1.41y(4)=441​≈1.41。因此在三等分时会使最后的结果达到最大值。

考虑有余数的情况：

  1. 余数 2：保留余数就好了，最后的结果是 3a×23^a \times 23a×2
  2. 余数 1：需要拆出一个 3，构成 2×22 \times 22×2，最后的结果是 3a−1×43^{a - 1} \times 43a−1×4
