338.比特位计数

发表于2021-03-22|更新于2021-03-22|LeetCode题解301-400

题目信息

题目描述

给定一个非负整数 num, 对于 0≤i≤num0 \le i \le num0≤i≤num范围中的每个数字 i, 计算其二进制数中的 1 的数目并将它们作为数组返回。

示例输出

输入： 2
输出： [0, 1, 1]

输入： 5
输出： [0, 1, 1, 2, 1, 2]

解题思路

不妨先列举几个十进制数的二进制表示（下标表示进制）：

0(10)=0(2)1(10)=1(2)2(10)=10(2)3(10)=11(2)4(10)=100(2)5(10)=101(2)6(10)=110(2)7(10)=111(2)\begin{aligned} 0_{(10)}&=0_{(2)}\\ 1_{(10)}&=1_{(2)}\\ 2_{(10)}&=10_{(2)}\\ 3_{(10)}&=11_{(2)}\\ 4_{(10)}&=100_{(2)}\\ 5_{(10)}&=101_{(2)}\\ 6_{(10)}&=110_{(2)}\\ 7_{(10)}&=111_{(2)} \end{aligned} 0(10)​1(10)​2(10)​3(10)​4(10)​5(10)​6(10)​7(10)​​=0(2)​=1(2)​=10(2)​=11(2)​=100(2)​=101(2)​=110(2)​=111(2)​​

可以看到在移除一个二进制数（ number ）的最低位后，它将和 ⌊number2⌋\lfloor \frac {number} 2 \rfloor⌊2number​⌋ 的二进制表示相同。对于奇数，1 的个数需要加 1；对于偶数，1 的个数是相同的。

根据这个结论，得到递推式：bitNumber[i]=bitNumber[⌊i2⌋]+(i%2)bitNumber[i] = bitNumber[\lfloor \frac i 2 \rfloor] + (i \% 2)bitNumber[i]=bitNumber[⌊2i​⌋]+(i%2)，其中 bitNumber 是一个保存二进制数的数组（除 2 和判断奇偶性均可通过位运算实现）。

尝试移除一下二进制数的高位：

4(10)=100(2)→00(2)=0(10)5(10)=101(2)→01(2)=1(10)6(10)=110(2)→10(2)=2(10)\begin{aligned} 4_{(10)}&=100_{(2)}\to 00_{(2)}=0_{(10)}\\ 5_{(10)}&=101_{(2)}\to 01_{(2)}=1_{(10)}\\ 6_{(10)}&=110_{(2)}\to 10_{(2)}=2_{(10)} \end{aligned} 4(10)​5(10)​6(10)​​=100(2)​→00(2)​=0(10)​=101(2)​→01(2)​=1(10)​=110(2)​→10(2)​=2(10)​​

对于 2n2^n2n 的数来说，移除最高位后一定是 0，因为这些数的二进制表示只有最高位是 1，其他位都是 0。对于其他的数，他们与转换后的数之间差了一个离他最近的 2n2^n2n 的举例，同时在 1 的个数上也差了 1 个。

根据这个结论，可以得到递推式：bitNumber[i]=bitNumber[i−distance]+1bitNumber[i] = bitNumber[i - distance] + 1bitNumber[i]=bitNumber[i−distance]+1，其中 distance=max(2n),n≤idistance = max(2^n), \enspace n \le idistance=max(2n),n≤i。
