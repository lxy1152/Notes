前缀和

发表于2021-03-02|更新于2021-04-13|LeetCode概念梳理

概念

前缀和是一个数组的某项下标之前（包括此项元素）的所有数组元素的和。

举例

对于某个一维数组 a、前缀和数组 b，根据前缀和的定义可以得到如下的关系式：

b[1]=a[1]b[2]=a[1]+a[2]=b[1]+a[2]b[3]=a[1]+a[2]+a[3]=b[2]+a[3]…b[i]=a[1]+a[2]+⋯+a[i]=b[i−1]+a[i]\begin{aligned} b[1] &= a[1]\\ b[2] &= a[1] + a[2] = b[1] + a[2]\\ b[3] &= a[1] + a[2] + a[3] = b[2] + a[3]\\ \dots \\ b[i] &= a[1] + a[2] + \dots + a[i] = b[i - 1] + a[i] \end{aligned} b[1]b[2]b[3]…b[i]​=a[1]=a[1]+a[2]=b[1]+a[2]=a[1]+a[2]+a[3]=b[2]+a[3]=a[1]+a[2]+⋯+a[i]=b[i−1]+a[i]​

根据上面的递推式可以得到任意某区间 [i,j][i, j][i,j] 的元素和为：a[i]+a[i+1]+⋯+a[j]=b[j]−b[i]+a[i]a[i] + a[i + 1] + \dots + a[j] = b[j] - b[i] + a[i]a[i]+a[i+1]+⋯+a[j]=b[j]−b[i]+a[i]。

但是需要注意的是，代码中的索引从 0 开始，而 b[0−1]=b[−1]b[0 - 1] = b[-1]b[0−1]=b[−1]，所以在索引 0 处需要做特殊处理，直接令 b[0] = a[0]。

除此之外也可以将前缀和数组统一向右移一位，使 b[i]=b[i−1]+a[i−1]b[i] = b[i - 1] + a[i - 1]b[i]=b[i−1]+a[i−1]，这样就可以避免索引 0 处的特殊处理了。

二维数组与一维数组类似，不同点在于元素和变成了一个平面区域内的所有元素的和。以 304.二维区域和检索-矩阵不可变 一题中的示例矩阵 matrix 来举例（行列均从 1 开始）：

3014256321120154101710305\def\arraystretch{1.5} \begin{array}{|c|c|c|c|c|} \hline 3 & 0 & 1 & 4 & 2 \\ \hline 5 & 6 & 3 & 2 & 1 \\ \hline 1 & 2 & 0 & 1 & 5 \\ \hline 4 & 1 & 0 & 1 & 7 \\ \hline 1 & 0 & 3 & 0 & 5 \\ \hline \end{array} 35141​06210​13003​42110​21575​​

如果希望求第二行第二列元素（ 6 ）的前缀和，那么 b[2,2]=a[1,1]+a[1,2]+a[2,1]+a[2,2]=14b[2, 2] = a[1, 1] + a[1, 2] + a[2, 1] + a[2, 2] = 14b[2,2]=a[1,1]+a[1,2]+a[2,1]+a[2,2]=14，即第一行第一列元素和当前元素所围成的区域的所有元素的和。

先从对角线上的元素尝试着开始推一下关系式：

b[1,1]=a[1,1]b[2,2]=a[1,1]+a[1,2]+a[2,1]+a[2,2]=b[1,2]+b[2,1]−b[1,1]+a[2,2]…b[n,n]=a[1,1]+a[1,2]+⋯+a[1,n]+a[2,1]+a[2,2]+a[2,n]+⋯+a[n,1]+a[n,2]+⋯+a[n,n]=b[n−1,n]+b[n,n−1]−b[n−1,n−1]+a[n,n]\begin{aligned} b[1, 1] &= a[1, 1]\\ b[2, 2] &= a[1, 1] + a[1, 2] + a[2, 1] + a[2, 2] = b[1, 2] + b[2, 1] - b[1, 1] + a[2, 2]\\ \dots \\ b[n, n] &= a[1, 1] + a[1, 2] + \dots + a[1, n] + a[2, 1] + a[2, 2] + a[2, n] + \dots + a[n, 1] + a[n, 2] + \dots + a[n, n]\\ &= b[n - 1, n] + b[n, n - 1] - b[n - 1, n - 1] + a[n, n] \end{aligned} b[1,1]b[2,2]…b[n,n]​=a[1,1]=a[1,1]+a[1,2]+a[2,1]+a[2,2]=b[1,2]+b[2,1]−b[1,1]+a[2,2]=a[1,1]+a[1,2]+⋯+a[1,n]+a[2,1]+a[2,2]+a[2,n]+⋯+a[n,1]+a[n,2]+⋯+a[n,n]=b[n−1,n]+b[n,n−1]−b[n−1,n−1]+a[n,n]​

b[n,n]b[n, n]b[n,n] 之所以可以用前几项来表示是因为可以把它理解为求一个正方形的面积，如下图所示。总的正方形面积等于红色框部分的面积和橙色框部分的面积之和，再加上蓝色框部分的面积。但由于红色框和橙色框部分有重复的部分，所以需要把这部分的面积减去。

回到上面的公式， b[n−1,n]b[n - 1, n]b[n−1,n] 代表红色框部分的面积，b[n,n−1]b[n, n - 1]b[n,n−1] 代表橙色框部分的面积，b[n−1,n−1]b[n - 1, n - 1]b[n−1,n−1] 代表相交部分的面积，a[n,n]a[n, n]a[n,n] 代表蓝色框部分的面积。把面积替换为元素和，就是前缀和了。

对于非对角线的情况，不妨假设 b[m,0]b[m, 0]b[m,0] 和 b[0,n]b[0, n]b[0,n] 均为0（m，n 为不超过 a 矩阵行列最大值的一个整数），那么上面的公式也是生效的。

综上，二维数组的前缀和递推式为（b[m,0]=b[0,n]=0b[m, 0] = b[0, n] = 0b[m,0]=b[0,n]=0）：

b[m,n]=b[m,n−1]+b[m−1,n]−b[m−1,n−1]+a[m,n]b[m, n] = b[m, n - 1] + b[m - 1, n] - b[m - 1, n - 1] + a[m, n]b[m,n]=b[m,n−1]+b[m−1,n]−b[m−1,n−1]+a[m,n]

进一步地可以得到任意两位置之间的元素和，假设第一个位置为 [row1, column1]，第二个位置为 [row2, column2]，并且 row1≤row2row1 \le row2row1≤row2，column1≤column2column1 \le column2column1≤column2，则：

a[row1,column1]+a[row1,column1+1]+⋯+a[row1,column2]+⋯+a[row2,column1]+a[row2,column1+1]+⋯+a[row2,column2]=b[row2,column2]−b[row1][column2]−b[row2][column1]+a[row1][column1]\begin{aligned} &a[row1, column1] + a[row1, column1 + 1] + \dots + a[row1, column2] + \dots + a[row2, column1] + a[row2, column1 + 1] + \dots + a[row2, column2] \\ &= b[row2, column2] - b[row1][column2] - b[row2][column1] + a[row1][column1] \end{aligned} ​a[row1,column1]+a[row1,column1+1]+⋯+a[row1,column2]+⋯+a[row2,column1]+a[row2,column1+1]+⋯+a[row2,column2]=b[row2,column2]−b[row1][column2]−b[row2][column1]+a[row1][column1]​

考虑到程序中数组从 0 开始，以上公式需要进行一定的修正（ result 代表任意两位置之间的元素和）：

b[m,n]=b[m,n−1]+b[m−1,n]−b[m−1,n−1]+a[m−1,n−1]result=b[row2+1,column2+1]−b[row1][column2+1]−b[row2+1][column1]+a[row1][column1]\begin{aligned} b[m, n] &= b[m, n - 1] + b[m - 1, n] - b[m - 1, n - 1] + a[m - 1, n - 1]\\ result &= b[row2 + 1, column2 + 1] - b[row1][column2 + 1] - b[row2 + 1][column1] + a[row1][column1] \end{aligned} b[m,n]result​=b[m,n−1]+b[m−1,n]−b[m−1,n−1]+a[m−1,n−1]=b[row2+1,column2+1]−b[row1][column2+1]−b[row2+1][column1]+a[row1][column1]​
