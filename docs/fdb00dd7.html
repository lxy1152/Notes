1807.替换字符串中的括号内容

发表于2021-03-29|更新于2021-04-01|LeetCode题解1801-1900

题目信息

题目描述

给你一个字符串 s，它包含一些括号对，每个括号中包含一个非空的键。

比如在字符串 "(name)is(age)yearsold" 中，有两个括号对，分别包含键 "name" 和 "age"。
你知道许多键对应的值，这些关系由二维字符串数组 knowledge 表示，其中 knowledge[i] = [key_i, value_i]，表示键 key_i 对应的值为 value_i。

你需要替换所有的括号对。当你替换一个括号对，且它包含的键为 key_i 时，你需要将 key_i 和括号用对应的值 value_i 替换。

如果从 knowledge 中无法得知某个键对应的值，你需要将 key_i 和括号用问号 "?" 替换（不需要引号）。

请你返回替换所有括号对后的结果字符串。

knowledge 中每个键最多只会出现一次，s 中不会有嵌套的括号。

  • 1<=s.length<=1051 <= s.length <= 10^51<=s.length<=105
  • 0<=knowledge.length<=1050 <= knowledge.length <= 10^50<=knowledge.length<=105
  • knowledge[i].length == 2
  • 1<=keyi.length,valuei.length<=101 <= key_i.length, value_i.length <= 101<=keyi​.length,valuei​.length<=10
  • s 只包含小写英文字母和圆括号 '(' 和 ')'
  • s 中每一个左圆括号 '(' 都有对应的右圆括号 ')'
  • s 中每对括号内的键都不会为空
  • s 中不会有嵌套括号对
  • key_i 和 value_i 只包含小写英文字母
  • knowledge 中的 key_i 不会重复

示例输出

输入： s = "(name)is(age)yearsold", knowledge = [["name", "bob"], ["age", "two"]]
输出： "bobistwoyearsold"

输入： s = "hi(name)", knowledge = [["a", "b"]]
输出： "hi?"

输入： s = "(a)(a)(a)aaa", knowledge = [["a", "yes"]]
输出： "yesyesyesaaa"

输入： s = "(a)(b)", knowledge = [["a", "b"], ["b", "a"]]
输出： "ba"

解题思路

没啥说的，先将键值保存到 HashMap 中，再逐个替换键。
